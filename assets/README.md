# Assets

Drop the following image files into this folder:

| File | Description |
|------|-------------|
| `press-photo.jpg` | Studio press photo (dark moody background) — used as hero background and About section photo |
| `starborn-logo.png` | Grunge-style $TARBORN wordmark logo (black on white) — CSS will invert it to white for the dark background |
| `ssab-logo.png` | $$AB "Some Stars Are Built Born" logo (black on white) — CSS will apply a gold tint |

After dropping the files in, commit and push. The site will automatically pick them up.
